# 📚 BookVerse — Ваша Личная Библиотека

**BookVerse** — это полнофункциональное мобильное приложение на Flutter для управления личной библиотекой. Приложение позволяет отслеживать книги, их статусы, управлять данными в оффлайне и анализировать вашу коллекцию с красивым Material 3 дизайном.

---

## 🎯 Основной Функционал

### 📖 **Управление Библиотекой**
- ✅ **Добавление книг** — создавайте новые записи о книгах с названием, автором и описанием
- ✅ **Редактирование** — изменяйте информацию об уже добавленных книгах
- ✅ **Удаление** — удаляйте книги с подтверждением
- ✅ **Просмотр деталей** — смотрите полную информацию о каждой книге
- ✅ **Статусы книг** — отмечайте статус: Доступна, Выдана, Зарезервирована, Недоступна

### 🔍 **Умный Поиск и Фильтрация**
- 🔎 **Живой поиск** — ищите по названию и автору в реальном времени
- 🏷️ **Фильтрация по статусу** — быстро находите книги по их статусу
- 🧹 **Очистка фильтров** — одна кнопка чтобы сбросить все фильтры
- 📊 **Отображение результатов** — видите количество найденных книг

### 🎨 **Персонализация**
- 🌓 **Темная/Светлая тема** — переключайтесь между темами одним нажатием
- 💾 **Сохранение настроек** — ваш выбор темы сохраняется между запусками
- 🔄 **Циклическое переключение** — Светлая → Тёмная → Система → Светлая

### 📤 **Экспорт Данных**
- 📊 **Экспорт в CSV** — скачайте список книг в электронную таблицу
- 📄 **Экспорт в PDF** — создайте красиво отформатированный документ
- 📱 **Автоматический шеринг** — поделитесь файлом через мессенджер, почту или облако
- 🔄 **Фильтрованный экспорт** — экспортируются только видимые в данный момент книги

### 📌 **Онбординг при Первом Запуске**
- 🎬 **Интерактивный тур** — 4 информативных слайда с объяснением функций
- ⏭️ **Навигация** — кнопки "Пропустить" и "Далее"
- 📍 **Индикатор прогресса** — визуальное отображение текущего слайда
- 🚀 **Одноразовый** — показывается только при первом запуске

### 🔔 **Локальные Уведомления**
- 💬 **Напоминания о книгах** — напоминание про книги в процессе чтения
- ✨ **Уведомление о новых книгах** — оповещение когда добавлены новые книги
- ⏰ **Запланированные уведомления** — установите напоминание на конкретное время
- 🔁 **Периодические уведомления** — повторяющиеся напоминания с заданным интервалом

### 📱 **Оффлайн-работа**
- 🗄️ **Локальная база данных (Hive)** — все данные хранятся на устройстве
- 🌐 **Полная функциональность без интернета** — приложение работает везде
- ⚡ **Быстрая работа** — доступ к данным без задержек
- 💾 **Автоматическое сохранение** — все изменения сохраняются тут же

## 🏗️ Архитектура Приложения

Приложение построено на принципах **Clean Architecture** с разделением на слои:

```
lib/
├── core/                      # Общие утилиты
│   ├── init_hive.dart        # Инициализация базы данных
│   └── preferences_provider.dart  # Провайдер SharedPreferences
│
├── data/                       # Слой данных
│   ├── datasources/           # Работа с локальным хранилищем
│   │   └── hive_local_data_source.dart
│   ├── models/                # Модели данных для Hive
│   │   ├── book_model.dart
│   │   └── genre_model.dart
│   ├── repositories/          # Реализация интерфейсов репозиториев
│   │   └── hive_book_repository.dart
│   ├── services/              # Бизнес-сервисы
│   │   ├── export_service.dart        # Экспорт в CSV/PDF
│   │   └── notification_service.dart  # Работа с уведомлениями
│   └── providers/             # Riverpod провайдеры для данных
│       └── book_providers.dart
│
├── domain/                     # Бизнес-логика
│   ├── entities/              # Основные сущности
│   │   ├── book.dart
│   │   └── genre.dart
│   └── repositories/          # Интерфейсы репозиториев
│       ├── book_repository.dart
│       └── genre_repository.dart
│
└── presentation/               # Слой представления (UI)
    ├── pages/                 # Экраны приложения
    │   ├── library_screen.dart    # Главный экран с библиотекой
    │   └── onboarding_screen.dart # Экран онбординга
    ├── viewmodels/            # Логика состояния
    │   ├── book_notifier.dart
    │   ├── book_view_state.dart
    │   ├── theme_notifier.dart
    │   └── onboarding_notifier.dart
    ├── providers/             # Riverpod провайдеры для UI
    │   ├── book_presentation_providers.dart
    │   ├── theme_provider.dart
    │   └── export_provider.dart
    └── widgets/               # Переиспользуемые компоненты
        └── book_card.dart     # Карточка книги
```

## 🚀 Как Начать Работу

### Требования
- Flutter 3.9.2 или выше
- Dart SDK совместимый с Flutter
- Android/iOS эмулятор или реальное устройство

### Установка

1. **Клонируйте проект**
```bash
cd bookvers
```

2. **Установите зависимости**
```bash
flutter pub get
```

3. **Запустите приложение**
```bash
flutter run
```

4. **Запустите тесты (опционально)**
```bash
flutter test
```

## 📖 Инструкция по Использованию

### 🎬 При Первом Запуске

1. **Просмотрите онбординг**
   - Нажимайте "Далее" для перелистывания слайдов
   - Или нажмите "Пропустить" чтобы перейти прямо в приложение
   - На последнем слайде нажмите "Начать"

2. **Главный экран появится**
   - Пустой список книг (нужно добавить первые книги)
   - FAB кнопка (🔴 красная кнопка внизу справа) для добавления

### ➕ Добавление Книги

1. **Нажмите на FAB кнопку** (+) внизу справа
2. **Заполните форму:**
   - Название книги
   - Имя автора
   - Описание (опционально)
   - Выберите статус (Доступна, Выдана, Зарезервирована, Недоступна)
3. **Нажмите "Сохранить"**
4. **Книга появится в списке**

### 🔎 Поиск и Фильтрация

1. **Поиск по названию/автору:**
   - Начните печатать в поле поиска
   - Результаты обновляются в реальном времени
   - Нажмите ❌ чтобы очистить

2. **Фильтрация по статусу:**
   - Нажимайте чипсы (Все, Доступна, Выдана, и т.д.)
   - Список фильтруется мгновенно
   - Нажмите "Очистить" чтобы сбросить все

3. **Комбинированный поиск:**
   - Введите текст поиска И выберите статус
   - Результаты будут соответствовать обоим критериям

### ✏️ Редактирование Книги

1. **Найдите книгу в списке**
2. **Нажмите на три точки (⋮) на карточке книги**
3. **Выберите "Редактировать"**
4. **Измените нужные поля**
5. **Нажмите "Обновить"**

### 🗑️ Удаление Книги

1. **Найдите книгу в списке**
2. **Нажмите на три точки (⋮) на карточке книги**
3. **Выберите "Удалить"**
4. **Подтвердите удаление в диалоговом окне**
5. **Книга удалится из библиотеки**

### 🎨 Смена Темы

1. **Нажмите на иконку луны/солнца** в верхнем правом углу
2. **Тема переключится**
3. **Или нажимайте многократно чтобы циклить:**
   - Светлая (солнце) ☀️
   - Тёмная (луна) 🌙
   - Система (по настройкам устройства) ⚙️

### 📤 Экспорт Данных

1. **Нажмите на иконку Download** (стрелка вниз) в верхнем правом углу
2. **Выберите формат:**
   - **"Экспорт в CSV"** — для Excel, таблиц
   - **"Экспорт в PDF"** — красивый документ

3. **Приложение создаст файл и откроет Share меню**
4. **Выберите куда отправить:**
   - 📧 Email
   - 💬 Мессенджер (WhatsApp, Telegram и т.д.)
   - ☁️ Облако (Google Drive, OneDrive и т.д.)
   - 💾 Сохранить локально

### 🔔 Уведомления

**Уведомления показываются автоматически:**

- 📚 Напоминание о книгах в процессе чтения
- ✨ Оповещение когда добавлены новые книги
- ⏰ Запланированные напоминания в установленное время

**Чтобы отключить уведомления:**
- На Android: Параметры → Приложения → BookVerse → Уведомления → Отключить
- На iOS: Параметры → Уведомления → BookVerse → Отключить

## 💾 Как Работает Локальное Хранилище

### Hive База Данных

1. **Инициализация при запуске:**
   - Регистрируются TypeAdapters для BookModel и GenreModel
   - Создаются нужные боксы (таблицы)
   - Данные загружаются из памяти устройства

2. **Сохранение данных:**
   - Каждое действие (добавление, изменение, удаление) сохраняется мгновенно
   - Нет необходимости в синхронизации с сервером
   - Данные шифруются на устройстве

3. **Оффлайн работа:**
   - Все операции работают без интернета
   - При наличии интернета можно добавить синхронизацию с сервером
   - Структура позволяет легко интегрировать API

### SharedPreferences

- Сохранение выбранной темы (light/dark/system)
- Статус завершения онбординга
- Простые настройки приложения

## 📝 Примеры Использования API

### Добавление книги программно
```dart
final book = const Book(
  id: 1,
  title: 'Война и мир',
  author: 'Лев Толстой',
  description: 'Исторический роман',
  status: BookStatus.available,
  genreId: 1,
);

ref.read(bookNotifierProvider.notifier).addBook(book);
```

### Поиск книг
```dart
ref.read(bookNotifierProvider.notifier).setSearchQuery('толстой');
final filtered = ref.watch(filteredBooksProvider);
```

### Смена темы
```dart
await ref.read(themeProvider.notifier).toggleTheme();
```

### Экспорт в PDF
```dart
final books = ref.watch(allBooksProvider);
await ref.read(exportProvider.notifier).exportToPDF(books.value ?? []);
```
