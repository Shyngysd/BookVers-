# 📚 BookVerse — Ваша Личная Библиотека

**BookVerse** — это полнофункциональное мобильное приложение на Flutter для управления личной библиотекой. Приложение позволяет отслеживать книги, их статусы, управлять данными в оффлайне и анализировать вашу коллекцию с красивым Material 3 дизайном.

---

## 🎯 Основной Функционал

### 📖 **Управление Библиотекой**
- ✅ **Добавление книг** — создавайте новые записи о книгах с названием, автором и описанием
- ✅ **Редактирование** — изменяйте информацию об уже добавленных книгах

- ✅ **Статусы книг** — отмечайте статус: Доступна, Выдана, Зарезервирована, Недоступна

## � Firebase Authentication (Новое!)

### Настройка Firebase

1. **Установите FlutterFire CLI**
```bash
dart pub global activate flutterfire_cli
```

2. **Настройте Firebase в проекте**
```bash
flutterfire configure
```

Этот скрипт:
- Создаст/свяжет Firebase проект
- Сгенерирует `firebase_options.dart` с вашими учётными данными
- Добавит конфигурацию для Android, iOS, Web, macOS

3. **Скопируйте сгенерированный файл**
Если `firebase_options.dart` не был перезаписан автоматически, скопируйте его в `lib/firebase_options.dart`

### Включение Email/Password Authentication в Firebase Console

1. Зайдите в [Firebase Console](https://console.firebase.google.com)
2. Выберите ваш проект
3. Перейдите в **Authentication** → **Sign-in method**
4. Нажмите на **Email/Password**
5. Включите **Email/Password** опцию
6. Сохраните изменения

## 💾 Как Работает Локальное Хранилище

### Hive База Данных

1. **Инициализация при запуске:**
   - Регистрируются TypeAdapters для BookModel и GenreModel
   - Создаются нужные боксы (таблицы)
   - Данные загружаются из памяти устройства

2. **Сохранение данных:**
   - Каждое действие (добавление, изменение, удаление) сохраняется мгновенно
   - Нет необходимости в синхронизации с сервером
   - Данные шифруются на устройстве

3. **Оффлайн работа:**
   - Все операции работают без интернета
   - При наличии интернета можно добавить синхронизацию с сервером
   - Структура позволяет легко интегрировать API

### SharedPreferences

- Сохранение выбранной темы (light/dark/system)
- Статус завершения онбординга
- Простые настройки приложения

```dart

```

```dart
final books = ref.watch(allBooksProvider);

```
